**자바스크립트 기초 문법 수업 정리 노트**

1. **데이터 타입**

   * **원시 자료형 (Primitive type)**: 값 자체를 저장하며 변경 불가능한 데이터 유형

     * **Number**: 숫자
     * **String**: 문자열
     * **null**: 값이 없음을 나타내는 특수한 값
     * **undefined**: 변수 선언 후 값이 할당되지 않았을 때 자동으로 설정되는 값
     * **Boolean**: 참(true) 또는 거짓(false)

   * **조합 자료형 (Reference type)**: 객체 형태로 데이터를 저장

     * **Object**: 일반 객체
     * **Array**: 배열
     * **Function**: 함수

   * **기본 자료형 예시**:

     ```javascript
     const num = 10;        // Number
     let str = "Alice";     // String
     const obj = {};        // Object
     ```

2. **원시 자료형 - Number**

   * 자바스크립트에서 **Number**는 숫자 값을 표현하며, 실수와 정수를 모두 포함.
   * 예시:

     ```javascript
     let age = 25;    // 정수
     let price = 19.99; // 실수
     ```

3. **String (문자열)**

   * 문자열은 따옴표( `"` 또는 `'` )로 둘러싸인 텍스트입니다.

   * **문자열 연결**: `+` 연산자를 사용하여 문자열을 결합

     ```javascript
     const firstName = "Tony";
     const lastName = "Stark";
     console.log(firstName + " " + lastName);  // Tony Stark
     ```

   * **템플릿 리터럴**: 문자열 내에서 변수를 삽입할 수 있는 문법

     ```javascript
     const firstName = "Tony";
     const lastName = "Stark";
     console.log(`Full name: ${firstName} ${lastName}`);  // Full name: Tony Stark
     ```

4. **Boolean (불리언)**

   * `true` 또는 `false` 값을 나타내는 자료형
   * 비교 연산에서 사용:

     ```javascript
     console.log(1 == 1);    // true
     console.log(2 == 1);    // false
     ```

5. **연산자**

   * **산술 연산자**: 기본적인 수학 연산 (더하기, 빼기 등)

     ```javascript
     let a = 5;
     let b = 2;
     console.log(a + b); // 7
     console.log(a - b); // 3
     ```

   * **비교 연산자**: 값들을 비교하고 불리언 값을 반환

     ```javascript
     console.log(5 == 5);   // true
     console.log(5 != 3);   // true
     ```

   * **논리 연산자**: `&&` (AND), `||` (OR), `!` (NOT)

     ```javascript
     console.log(true && false);   // false
     console.log(true || false);   // true
     ```

6. **조건문**

   * **if 문**: 조건에 따라 실행되는 코드

     ```javascript
     let age = 18;
     if (age >= 18) {
       console.log("성인입니다.");
     }
     ```

   * **else if, else 문**:

     ```javascript
     let score = 85;
     if (score >= 90) {
       console.log("A");
     } else if (score >= 80) {
       console.log("B");
     } else {
       console.log("C");
     }
     ```

7. **반복문**

   * **while** 문: 조건이 참일 동안 반복

     ```javascript
     let i = 0;
     while (i < 5) {
       console.log(i);
       i++;
     }
     ```

   * **for** 문: 반복 횟수를 미리 정할 때 사용

     ```javascript
     for (let i = 0; i < 5; i++) {
       console.log(i);
     }
     ```

   * **for...in**: 객체의 속성 반복

     ```javascript
     const person = { name: "John", age: 30 };
     for (let key in person) {
       console.log(key, person[key]);
     }
     ```

   * **for...of**: 배열이나 문자열의 값을 반복

     ```javascript
     const numbers = [1, 2, 3];
     for (let num of numbers) {
       console.log(num);
     }
     ```

8. **함수 정의**

   * **함수 선언문**: 전통적인 방식

     ```javascript
     function greet(name) {
       return "Hello, " + name;
     }
     ```

   * **화살표 함수**: 간결한 문법

     ```javascript
     const greet = (name) => "Hello, " + name;
     ```

9. **기본 매개변수 (Default function parameter)**

   * 매개변수에 기본값을 설정할 수 있음

     ```javascript
     const greet = (name = "Anonymous") => "Hello, " + name;
     ```

10. **나머지 매개변수 (Rest parameter)**

    * 여러 개의 매개변수를 배열로 받아올 때 사용

    ```javascript
    const sum = (...numbers) => {
      return numbers.reduce((acc, num) => acc + num, 0);
    };
    console.log(sum(1, 2, 3, 4));  // 10
    ```

11. **스프레드 문법 (Spread syntax)**

    * 배열이나 객체를 풀어서 개별 요소로 나눠서 사용할 때 유용

    ```javascript
    const numbers = [1, 2, 3];
    const newNumbers = [...numbers, 4, 5];
    console.log(newNumbers);  // [1, 2, 3, 4, 5]
    ```

12. **NaN (Not a Number) 예시**

    * 숫자가 아닌 값을 연산할 때 발생

    ```javascript
    let result = "string" * 2;
    console.log(result);  // NaN
    ```

13. **null & undefined**

    * **null**: 값이 없다는 의도를 명시적으로 표현
    * **undefined**: 값이 할당되지 않은 변수

    ```javascript
    let a = null;
    let b;
    console.log(a);  // null
    console.log(b);  // undefined
    ```

**핵심 요약**:

* **기본 자료형**: `Number`, `String`, `Boolean`, `null`, `undefined` 등을 사용하여 데이터를 다룸.
* **연산자**와 **조건문**을 활용해 로직을 제어하고, **반복문**을 통해 효율적인 반복 작업을 처리.
* **함수**를 정의하고 다양한 방식으로 매개변수를 처리하며, **화살표 함수**와 **스프레드 문법**으로 코드를 간결하게 작성할 수 있음.
