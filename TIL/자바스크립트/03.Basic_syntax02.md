**자바스크립트 객체 및 배열 수업 정리 노트**

### 1. **객체 (Object)**

* **객체**는 **키-값 쌍**으로 데이터를 저장하는 자료형으로, 데이터의 구조를 설정할 때 사용됩니다.
* **구조**:

  * `key: value` 형식으로 속성 정의.
  * 예시:

    ```javascript
    const person = {
      name: "Alice",
      age: 30,
      greet: function() {
        return "Hello, " + this.name;
      }
    };
    ```
* 객체 속성에 접근할 때는 **dot notation** 또는 **bracket notation**을 사용합니다.

  ```javascript
  console.log(person.name);  // Alice
  console.log(person['age']);  // 30
  ```

### 2. **메서드 (Method)**

* 객체 내에서 **함수**를 메서드라고 하며, 메서드를 사용하여 객체의 동작을 정의합니다.
* 메서드를 정의할 때 함수처럼 `function` 키워드를 사용할 수 있으며, **`this`** 키워드를 이용해 객체 자신을 참조할 수 있습니다.

  ```javascript
  const person = {
    name: "Alice",
    greet: function() {
      console.log("Hello, " + this.name);
    }
  };
  person.greet();  // Hello, Alice
  ```

### 3. **this 키워드**

* `this`는 현재 실행되고 있는 **메서드의 컨텍스트**를 참조하는데 사용됩니다.
* 예시:

  ```javascript
  const person = {
    name: "Alice",
    greet: function() {
      console.log("Hello, " + this.name); // this는 person 객체를 참조
    }
  };
  person.greet();  // Hello, Alice
  ```
* **this**는 호출되는 **컨텍스트에 따라** 다르게 작동할 수 있습니다. 예를 들어, 일반 함수에서 `this`는 **전역 객체**를 가리킬 수 있습니다.

### 4. **추가 객체 문법**

* **구조 분해 할당 (Destructuring Assignment)**: 객체나 배열을 분해하여 값을 추출하는 방법.

  ```javascript
  const person = { name: "Alice", age: 30 };
  const { name, age } = person;
  console.log(name);  // Alice
  console.log(age);  // 30
  ```

* **계산된 속성 이름 (Computed Property Name)**: 객체의 속성 이름을 동적으로 생성할 수 있습니다.

  ```javascript
  const key = "name";
  const person = {
    [key]: "Alice"
  };
  console.log(person.name);  // Alice
  ```

* **스프레드 문법 (Spread Syntax)**: 객체나 배열을 풀어서 다른 객체나 배열에 복사하거나 결합할 때 사용됩니다.

  ```javascript
  const person = { name: "Alice", age: 30 };
  const updatedPerson = { ...person, city: "New York" };
  console.log(updatedPerson);  // { name: 'Alice', age: 30, city: 'New York' }
  ```

* **Optional chaining (`?.`)**: 객체의 속성에 접근할 때 값이 없을 경우 `undefined`를 반환하도록 하여 오류를 방지합니다.

  ```javascript
  const user = { name: "Alice", address: null };
  console.log(user.address?.street);  // undefined
  ```

### 5. **JSON (JavaScript Object Notation)**

* **JSON**은 자바스크립트 객체를 표현하기 위한 **문자열 형식**으로 데이터를 주고받을 때 많이 사용됩니다.
* JSON과 객체는 서로 변환할 수 있습니다:

  * `JSON.stringify()` - 객체를 JSON 문자열로 변환.
  * `JSON.parse()` - JSON 문자열을 객체로 변환.

  ```javascript
  const person = { name: "Alice", age: 30 };
  const jsonString = JSON.stringify(person);
  console.log(jsonString);  // '{"name":"Alice","age":30}'
  const jsonObject = JSON.parse(jsonString);
  console.log(jsonObject.name);  // Alice
  ```

### 6. **배열 (Array)**

* **배열**은 순서가 있는 데이터 구조로, 여러 개의 값을 저장할 수 있습니다.

* 배열의 **기본 구조**:

  ```javascript
  const names = ["Alice", "Bob", "Charlie"];
  console.log(names.length);  // 3
  ```

* **배열 메서드**:

  * `push()`: 배열의 끝에 요소를 추가.
  * `pop()`: 배열의 마지막 요소를 제거.
  * `unshift()`: 배열의 시작 부분에 요소를 추가.
  * `shift()`: 배열의 첫 번째 요소를 제거.

  ```javascript
  names.push("Dave");
  names.pop();
  names.unshift("Eve");
  names.shift();
  ```

### 7. **배열 순회 메서드**

* **forEach()**: 배열의 각 요소에 대해 함수를 실행.

  ```javascript
  names.forEach(function(name) {
    console.log(name);
  });
  ```

* **map()**: 배열의 각 요소를 변환하여 새로운 배열을 반환.

  ```javascript
  const numbers = [1, 2, 3];
  const squared = numbers.map(num => num * num);
  console.log(squared);  // [1, 4, 9]
  ```

### 8. **배열과 전개 구문 (Spread Syntax)**

* 배열에서 스프레드 문법을 사용하여 배열을 합치거나 복사할 수 있습니다.

  ```javascript
  const parts = ["engine", "wheels"];
  const newParts = [...parts, "brakes"];
  console.log(newParts);  // ["engine", "wheels", "brakes"]
  ```

**핵심 요약**:

* **객체**는 키-값 쌍으로 데이터를 저장하고, **메서드**를 통해 동작을 정의합니다.
* **`this`** 키워드를 통해 객체 내에서 자신을 참조할 수 있으며, **optional chaining**으로 안전하게 객체를 다룰 수 있습니다.
* **배열**은 순서가 있는 데이터 구조로, 다양한 **배열 메서드**를 사용하여 요소를 추가/제거하거나 변환할 수 있습니다.
* **JSON**은 객체와 문자열 간 변환을 통해 데이터를 주고받는 데 사용됩니다.
